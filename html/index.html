What is the Karate Tournament Utility all about?

This Karate tournament utility was built to ease the process of a karate tournament. It aims to take out human error and making data storage a lot eaiser. System admins will be appointed to trained users in every dojo that ends up using this system. Every action will be tracked if data is tampered with. These sepcific people will be trained in how to work the website and enter data into the database. The utility is also for the public. With public use everything except for the entering of data will work. So, please enjoy, have fun, and train hard. If any questions need to be answered, there is a contact page in the taskbar to be able to get ahold of me.
